# 👗 Multi-Modal Fashion Trend Prediction

## 📌 Overview
This project presents a **multi-modal approach for fashion trend prediction** by integrating **image, textual, and time-series data** using machine learning and deep learning techniques.

The system combines:
- CNN for image analysis  
- Sentiment Analysis for customer reviews  
- LSTM for time-series prediction  
- Ensemble Learning for final trend prediction  

---

## 🚀 Features
- 🔍 Image feature extraction using CNN  
- 💬 Sentiment analysis using TF-IDF + Logistic Regression  
- 📈 Time-series prediction using LSTM  
- 🤖 Ensemble model for improved accuracy  
- 📊 Visualization of model performance  

---

## 🧠 Methodology

### 1. Image Processing (CNN)
Extracts visual features such as:
- Style  
- Texture  
- Color  

### 2. Text Processing (Sentiment Analysis)
- TF-IDF vectorization  
- Logistic Regression classification  
- Detects positive/negative sentiment  

### 3. Time-Series Analysis (LSTM)
- Captures temporal patterns  
- Predicts future trends  

### 4. Ensemble Model
- Combines all model outputs  
- Uses weighted averaging  
- Improves overall performance
