Multi-Modal Fashion Trend Prediction using CNN, Sentiment Analysis & LSTM

This research proposes a multi-modal approach for fashion trend prediction by integrating image, textual, and time-series data using machine learning and deep learning techniques.

Problem Statement

Existing fashion trend prediction methods rely on single-source data such as images or textual reviews, leading to limited accuracy. These approaches fail to capture the combined influence of visual features, customer sentiment, and temporal patterns.

Literature Review

CNN models extract image features, NLP techniques analyze sentiment, and LSTM models capture temporal patterns. However, most methods use single data modalities.

Research Gap

There is limited research integrating image, text, and time-series data into a unified system for accurate trend prediction.

Methodology

CNN

Extracts visual features such as style, texture, and color from images.

Sentiment Analysis

TF-IDF + Logistic Regression used to classify customer reviews.

LSTM

Captures temporal patterns from time-series data.

Ensemble Model

Combines outputs of CNN, Sentiment, and LSTM using weighted averaging.

Results

The ensemble model achieves strong performance with high recall and balanced accuracy, demonstrating effective multi-modal learning.

Conclusion

The proposed system effectively integrates visual, textual, and temporal data for fashion trend prediction. The ensemble approach improves accuracy and provides more reliable predictions compared to individual models.
